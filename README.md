# 🧮 Calculadora de IMC com Orientação a Objetos

Este projeto foi desenvolvido como parte dos meus estudos de **JavaScript** na plataforma DIO. O objetivo principal foi praticar conceitos de **Programação Orientada a Objetos (POO)**.

## 🚀 Tecnologias Utilizadas
* JavaScript (ES6+)
* Node.js (para execução via terminal)
* Git & GitHub

## 📝 Funcionalidades
* Criação de instâncias de pessoas através de uma `class`.
* Cálculo automático de IMC via método interno.
* Classificação do status de saúde (Abaixo do peso, Normal, Sobrepeso, etc.).

## 💻 Como rodar o projeto
1. Clone o repositório.
2. Abra o terminal na pasta do projeto.
3. Digite: `node seu_arquivo.js` (substitua pelo nome do seu arquivo).
